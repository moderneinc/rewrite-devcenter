# Samples

Generated artifacts that demonstrate the Moderne EOL feed format and what the cards produce. They
are derived from the live [endoflife.date](https://endoflife.date) service and checked in for
offline review. Regenerate everything with:

```bash
./gradlew generateSamples
```

## `feeds/`

Per-ecosystem snapshots of the bundled feed in the [Moderne EOL feed format](../FEED-FORMAT.md),
filtered by coordinate type:

| File                       | Contents                                              |
|----------------------------|-------------------------------------------------------|
| `eol-feed.jvm.yaml`        | Products with `pkg:maven/...` coordinates             |
| `eol-feed.npm.yaml`        | Products with `pkg:npm/...` coordinates               |
| `eol-feed.nuget.yaml`      | Products with `pkg:nuget/...` coordinates             |
| `eol-feed.runtime.yaml`    | Runtime products with `pkg:generic/...` coordinates   |
| `eol-feed.docker.yaml`     | Products with `pkg:docker/...` coordinates            |

These double as ready-to-edit starting points for an organization's own feed: copy one, add your
internal products, and point a card's `feed` option at it.

## `extracts/`

Example `EndOfLifeReport` output, one pair of files per card, generated by running the card against
a representative fixture with the **bundled feed** and a pinned `asOf` date (see
`SampleExtractsTest`). Each `*.ndjson` is one JSON object per detected item; each `*.csv` is the
same data with a header row.

| File                              | Card                          | Fixture                                   |
|-----------------------------------|-------------------------------|-------------------------------------------|
| `jvm-end-of-life.*`               | `DependencyEndOfLife`         | a POM using Spring Boot 2.7.18            |
| `npm-end-of-life.*`               | `NpmDependencyEndOfLife`      | a `package.json` with express & vue       |
| `nuget-end-of-life.*`             | `NuGetDependencyEndOfLife`    | a `.csproj` referencing bootstrap 4.6.0   |
| `runtime-end-of-life.*`           | `RuntimeEndOfLife`            | Java 8, Node `>=16`, .NET `net6.0`, Python `3.8.18` |
| `docker-end-of-life.*`            | `DockerImageEndOfLife`        | a Dockerfile with `node:16.20-alpine`, `python:3.8-slim` |
